Smart Task Extractor is a lightweight NLP-based tool that converts natural 
language text into structured task data.

It extracts:
- Task name (what needs to be done)
- Deadline (when it is due)
- Priority (based on urgency)

The system is fully rule-based and uses regex + date parsing (no ML models required),
 making it fast, explainable, and easy to extend.

=============================
 Input Text
   ↓
Task Extraction (regex-based parsing)
   ↓
Deadline Extraction (dateparser NLP library)
   ↓
Priority Calculation (rule-based urgency scoring)
   ↓
Structured Output (JSON + CLI display)


====================================
This project was designed to prioritise:
- Simplicity (no ML dependencies)
- Explainability (every decision is traceable)
- Extensibility (easy to modify rules or add features)
- Performance (lightweight, fast execution)
===================

Design Decisions:

1. Rule-based NLP instead of machine learning
   → Ensures explainability and zero training overhead

2. External library (dateparser) used only for date understanding
   → Keeps core logic fully controllable

3. Modular pipeline architecture
   → Each stage can be replaced or upgraded independently

4. Config-driven urgency system
   → Easy to tune without touching core logic
